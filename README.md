# QR Maker

A lightweight, dependency-free QR code generator built with vanilla HTML, CSS, and JavaScript.

## Overview

QR Maker lets you generate customizable QR codes entirely in the browser — no server, no npm packages, no build step required. The QR encoding logic is inlined directly into the app using [nayuki's QR Code generator](https://github.com/nayuki/QR-Code-generator) (MIT license).

## Features

- Generate QR codes from URLs, plain text, WiFi credentials, and more
- Customize foreground/background colors
- Choose error correction level (L / M / Q / H)
- Export as PNG or SVG
- Share configurations via URL

## Tech Stack

- HTML5 Canvas
- Vanilla CSS
- Vanilla JavaScript (no dependencies)
